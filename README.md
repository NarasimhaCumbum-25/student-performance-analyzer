# student-performance-analyzer

##  Project Overview
The **Student Performance Analytics System** is a Python-based data analysis project developed using **Pandas**.  
It analyzes student academic performance by combining student details, subject marks, and attendance data to generate meaningful insights.

This project is designed to demonstrate **real-world data analysis skills** suitable for academic projects, resumes, and entry-level data roles.
